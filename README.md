# VRChat Config Editor

## 概要

VRChat Config Editorは、VRChatの設定ファイルを簡単に編集できるツールです。  
GUI版とCLI版の両方が提供されており、ユーザーの好みに応じて使用できます。

## 開発用メモ

[VRChatドキュメント configureation File](https://docs.vrchat.com/docs/configuration-file) を参考に該当設定ファイルのjsonを書き換えることで実現します。
Windows環境外で動作する場合も考え、configファイルのpathは必要に応じて変更できるようにします。
